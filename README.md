# Фінальний проект з алгоритмів

## Завдання 1: Однозв'язний список
- Реверсування списку через зміну посилань між вузлами
- Сортування злиттям (Merge Sort) O(n log n)
- Об'єднання двох відсортованих списків

## Завдання 2: Дерево Піфагора
- Рекурсивна візуалізація фрактала
- Користувач вказує рівень рекурсії (рекомендовано 5-12)

## Завдання 3: Алгоритм Дейкстри
- Пошук найкоротших шляхів у зваженому графі
- Оптимізація через бінарну купу (heapq)
- Складність: O((V + E) log V)

## Завдання 4: Візуалізація бінарної купи
- Побудова дерева з масиву купи
- Візуалізація через networkx та matplotlib

## Завдання 5: Обхід бінарного дерева
- DFS (в глибину) - використовує стек
- BFS (в ширину) - використовує чергу
- Градієнт кольорів від темного до світлого

## Завдання 6: Жадібний алгоритм та динамічне програмування
- Жадібний: вибір за співвідношенням калорії/вартість
- ДП: оптимальний розв'язок задачі про рюкзак

## Завдання 7: Метод Монте-Карло

### Аналіз збіжності

| Кількість кидків | Середня похибка |
|------------------|-----------------|
| 100 | ~1.56% |
| 1,000 | ~0.94% |
| 10,000 | ~0.15% |
| 100,000 | ~0.06% |
| 1,000,000 | ~0.02% |

### Висновки

1. **Підтвердження теоретичних розрахунків**: Результати методу Монте-Карло практично збігаються з аналітичними ймовірностями. Середня похибка при 1,000,000 симуляцій становить лише 0.025%.

2. **Закон великих чисел**: Зі збільшенням кількості симуляцій похибка зменшується. При 100 кидках похибка ~1.5%, при 1,000,000 — менше 0.03%.

3. **Розподіл ймовірностей**: Сума 7 має найвищу ймовірність (16.67%), оскільки існує 6 комбінацій для її отримання: (1,6), (2,5), (3,4), (4,3), (5,2), (6,1). Розподіл симетричний відносно 7.

4. **Практичне застосування**: Метод Монте-Карло є ефективним інструментом для перевірки теоретичних розрахунків та моделювання випадкових процесів, особливо коли аналітичний розрахунок складний або неможливий.
